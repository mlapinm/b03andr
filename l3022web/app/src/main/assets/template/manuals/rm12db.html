На курсе по основам SQL вы научитесь работать с такими популярными СУБД как SQLite, MySQL и PostgreSQL.

На этом уроке мы рассмотрим: создание базы данных, создание и удаление таблицы, вставка, вывод, удаление и изменение записей.

    Инструменты
  1. Терминал(cmd, git-bush, conemu)
  2. СУБД(SQLite, MySQL, PostgreSQL)
  3. Текстовый редактор (notepad, gedit, sublime)
    Установка Debian/Ubuntu
  • $sudo apt-get update
  • $sudo apt-get install sqlite3 mysql-server postgresql

    select * from citys limit 2;

    select * from citys order by id limit 2;

    select * from citys where id=2;

    select * from citys where id!=2;

    select * from citys where id>2 and id<=4 ;

    select * from citys where id<=2 or id>=4 ;

    select * from citys order by id asc;

    select * from citys order by id desc;

    update citys set name='Moscow' where id=2;

    delete from citys where name='Belorechensk';

    select count(*) from citys;

    quit
